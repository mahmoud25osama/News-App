# News-App

## Setup

Create a `config.js` file in the root directory with the following structure:

```javascript
const CONFIG = {
  GNEWS_API_KEY: "your-gnews-api-key",
  ALLSPORTSAPI_KEY: "your-allsportsapi-key",
  OPENWEATHER_API_KEY: "your-openweathermap-api-key",
  EXCHANGERATE_API_KEY: "your-exchangerate-api-key",
  DEFAULT_LEAGUE_ID: 152
};
```

Get your API keys from:
- [GNews](https://gnews.io/)
- [AllSportsAPI](https://allsportsapi.com/)
- [OpenWeatherMap](https://openweathermap.org/api)
- [ExchangeRate-API](https://www.exchangerate-api.com/)
